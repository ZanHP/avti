# Registrirani avtomobili v Sloveniji leta 2016

Projekt se nahaja v datoteki projekt.ipynb.

## Zajeti podatki za posamezen avtomobil:
- registrsko območje
- masa
- znamka
- prostornina motorja
- moč motorja
- vrsta goriva
- najvišja hitrost
- izpust CO2
- poraba goriva
- okoljevarstvena oznaka
- leto izdelave

## Podatki so analizirani glede na:
- registrsko območje
- "ljudske" predpostavke
- znamko
